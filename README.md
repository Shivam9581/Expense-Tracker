# Expense Tracker

A simple Expense Tracker web application that helps users manage their daily expenses and track their spending habits.

## Features

* Add new expenses
* View all recorded expenses
* Delete expenses
* Track total spending
* Simple and responsive user interface
* Data stored locally in the browser

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

Expense-Tracker/

├── index.html

├── style.css

├── script.js

└── README.md

## How to Run

1. Clone the repository:

   git clone https://github.com/Shivam9581/Expense-Tracker.git

2. Open the project folder.

3. Open `index.html` in your browser or run it using Live Server in VS Code.
